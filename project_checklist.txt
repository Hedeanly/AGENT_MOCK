API ENDPOINTS
-------------
GET    /listings         - Get all listings
GET    /listings/{id}    - Get one listing
POST   /listings         - Add new listing (admin only)
PUT    /listings/{id}    - Edit listing (admin only)
DELETE /listings/{id}    - Delete listing (admin only)
POST   /auth/login       - Admin login, returns JWT token
POST   /enquiries        - Submit enquiry form
GET    /enquiries        - View all enquiries (admin only)


TECH STACK
----------
Frontend : Plain HTML + CSS + JavaScript
Backend  : Python (FastAPI)
Database : PostgreSQL
Hosting  : Netlify (frontend) + Render/Railway (backend + DB)
